## Термины

| ПО, Программное обеспечение | ПО, Программное обеспечение программа для ЭВМ ТриД — Думай, Делай, Достигай. |
| --- | --- |
| Клиентская часть | Клиентская часть - программный компонент доступный через браузер, предназначенный для предоставления пользователю интерфейса взаимодействия с функциональными возможностями ПО. |
| Браузер | Google Chrome, Яндекс Браузер, Microsoft Edge (ранее Internet Explorer), Opera, Mozilla Firefox. |
| Курс | Базовый элемент обучающего контента в ТриД. Представляет собой структурированный материал по определённому объекту знаний, включающий набор слайдов (минимум один) с механизмом контроля прохождения пользователем и настройками доступности и назначения пользователям. |
| Слайд | Минимальный необходимый элемент курса. Может включать текст, изображения, видео, тестовые вопросы, а также другие поддерживаемые форматы контента в любой последовательности на усмотрение автора. Автор курсов в ТриД имеет возможность создавать и редактировать контент внутри курса по слайдам. |
| Мастерская курсов | Набор инструментов для создания, редактирования, назначения и управления курсами в ТриД. |
| Организация | Отдельный пользовательский интерфейс для пользователей (сотрудников) организации, для которых действуют общие политики настроек доступа к курсам своей организации и к курсам внешних организаций. Пространство, контент и сотрудники одной организации отделены от других организаций в ТриД. |
